## How to setup

First, clone this project: `git clone https://github.com/alaminsheikh01/Flight-Search-Project.git`

```bash
cd <project-name>

npm install

npm run dev
# or
yarn dev
```

Make .env.local in the root directory 

```
need to add api key:
NEXT_PUBLIC_API_KEY=<Sent in the email>
```
